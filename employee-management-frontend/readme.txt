npx create-react-app employee-management --template typescript
cd employee-management-frontend
npm install axios react-router-dom@6 tailwindcss




Backend Setup:
    Create the CRUD API endpoints using NestJS and TypeORM.
    Set up entities, services, and controllers for employee operations.
    Integrate Swagger for API documentation.

Frontend Setup:
    Install React, TypeScript, and Tailwind.
    Build the necessary components (EmployeeList, EmployeeForm).
    Use axios to connect the frontend to the backend API.

API Calls:
    Fetch all employees, add, update, and delete employees using Axios and display the result in the frontend.
